Pyramid Patterns in C

Pyramid Patterns:

A pyramid pattern is a pattern of numbers or characters that form a pyramid shape. For example, a pyramid pattern of numbers can be created by printing numbers in a triangular shape.

Program Description

Write a C Program to print pyramid patterns of numbers.

1. Take the number of rows as input.


2. Use nested loops to print the pyramid pattern.


3. Print the output and exit.

Here is source code of the C program to print pyramid patterns of numbers. The C program is successfully compiled and run on a Linux system. The program output is also shown below.


                /*

C Program to print pyramid patterns of numbers */
#include <stdio.h>

int main() { 
int rows, i, j, k;
 printf("Enter the number of rows: ");
scanf("%d", &rows);
for (i = 1; i <= rows; i++) {
    for (j = 1; j <= i; j++) {
        printf("%d ", j);
    }
    printf("\n");
}

return 0;
}
