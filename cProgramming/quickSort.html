Quick Sort in C

Quick Sort:

Quick Sort is a Divide and Conquer algorithm. It picks an element as pivot and partitions the given array around the picked pivot.

Program Description

Write a C Program to sort an array of integers using Quick Sort.

1. Take the array of integers as input.


2. Implement a partition function that takes the last element as pivot, places the pivot element at its correct position in sorted array, and places all smaller (smaller than pivot) to left of pivot and all greater elements to right of pivot.


3. Use recursion to sort the elements before and after the pivot.


4. Print the sorted array and exit.
