C Program to Find Sum of Digits of a Number

Sum of Digits:

The sum of digits of a number is the sum of all the digits present in the number. For example, the sum of digits of the number 1234 is 1+2+3+4 = 10.

Program Description

Write a C Program to find the sum of digits of a given number.

1. Take the integer to be checked as input.


2. Initialize a variable to store the sum of digits.


3. Use a while loop to extract each digit from the number and add it to the sum.


4. Print the output and exit.

Here is source code of the C program to find the sum of digits of a given number. The C program is successfully compiled and run on a Linux system. The program output is also shown below.


                /*

C Program to find the sum of digits of a number */
#include <stdio.h>

int main() { int n, sum = 0; printf("Enter a number: "); scanf("%d", &n);
while (n != 0) {
    sum += n % 10;
    n /= 10;
}

printf("Sum of digits: %d", sum);

return 0;
}
